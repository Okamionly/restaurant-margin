---

## Fix appliqué

**Fichier concerné** : `client/node_modules/` (environnement, pas de code modifié)  
**Commande** : `cd client && npm install`  
**Lock file mis à jour** : `client/package-lock.json`

**Avant** : `npx tsc --noEmit` → 30+ erreurs TS2307/TS7026  
**Après** : `npx tsc --noEmit` → 0 erreurs
