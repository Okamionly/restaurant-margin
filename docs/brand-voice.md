# RestauMargin -- Guide de Voix de Marque

## Positionnement

- **Cible** : restaurateurs independants (bistrots, brasseries, gastro, hotels-restaurants)
- Pas les chaines (c'est Inpulse)
- Prix accessible : 29EUR/mois
- IA qui agit, pas qui predit seulement

## Tonalite

- **Directe** -- pas de jargon corporate, on parle comme un chef
- **Confiante** -- "On gere vos marges" pas "On essaie de vous aider"
- **Professionnelle mais accessible** -- pas snob, pas trop casual
- **Orientee resultat** -- chiffres, economies, marges, pas des promesses vagues

## Mots a utiliser

- "Marge", "rentabilite", "cout matiere"
- "En cuisine", "votre carte", "vos fournisseurs"
- "IA", "automatique", "en temps reel"
- "Simple", "rapide", "29EUR/mois"

## Mots a eviter

- "Revolutionnaire", "disruptif" (trop startup)
- "Solution" (trop corporate)
- "Gratuit" (on n'a pas de plan gratuit)
- "Chaine", "franchise" (c'est la cible d'Inpulse)

## Exemples de copy

- **Landing hero** : "Maitrisez vos marges. Augmentez vos profits."
- **CTA** : "Voir les tarifs" (pas "Essai gratuit")
- **Chat IA** : "Je suis votre assistant RestauMargin. Dites-moi ce que vous voulez faire."
- **Email** : "Bonjour Chef," (pas "Cher client")
- **Erreur** : "Quelque chose s'est mal passe. On s'en occupe." (pas "Error 500")

## Guidelines visuelles

- **Couleur accent** : Teal (`#0d9488`)
- **Font titres** : Satoshi
- **Font corps** : General Sans / DM Sans
- **Ton visuel** : Luxury tech minimal (comme Linear)
- **Icones** : Lucide React, style outline
