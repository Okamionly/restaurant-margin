## Notes d'utilisation

### Variables a personnaliser
- `{{Prenom}}` : Prenom du restaurateur
- `{{NomRestaurant}}` : Nom de l'etablissement
- `{{Source}}` : Ou on a trouve le restaurant (Google Maps, TripAdvisor, LinkedIn, salon, etc.)

### Bonnes pratiques
- **Objet court** : < 50 caracteres, pas de majuscules, pas de points d'exclamation
- **Envoyer le mardi ou jeudi** entre 10h et 11h (hors service)
- **Ne jamais envoyer le vendredi, samedi ou dimanche** (service)
- **Follow-up a J+3** si pas de reponse (version 2 -- plus courte)
- **Follow-up a J+7** si toujours pas de reponse (dernier email, valeur pure)
- **Volume** : max 50 emails/jour pour eviter les filtres spam
- **Personnalisation** : toujours mentionner le nom du restaurant et un detail specifique

### A/B testing
- Tester les 3 versions d'objet sur 3 lots de 50 contacts
- Garder la version avec le meilleur taux d'ouverture
- Iterer sur le corps du mail avec le meilleur objet
