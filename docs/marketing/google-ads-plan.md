## 6. Conversion Tracking

### Evenement principal : sign_up

Le tag GA4 (G-656H54T9RS) est deja installe dans `client/index.html`.

L'evenement de conversion est declenche dans `client/src/pages/Login.tsx` lors d'une inscription reussie :

```javascript
// Declenche automatiquement apres register() reussi
gtag('event', 'sign_up', { method: 'email' });
```

### Configuration dans Google Ads

1. **Lier Google Ads a GA4** : Dans GA4, Admin > Google Ads Linking
2. **Importer la conversion** : Dans Google Ads, Outils > Conversions > Importer depuis GA4
3. **Selectionner** : Evenement `sign_up`
4. **Valeur** : 29 EUR (prix de l'abonnement mensuel)
5. **Modele d'attribution** : Data-driven (recommande) ou dernier clic

### Evenements supplementaires a configurer

| Evenement | Declencheur | Valeur |
|-----------|-------------|--------|
| sign_up | Inscription reussie | 29 EUR |
| trial_start | Debut essai gratuit | 10 EUR |
| page_view (pricing) | Visite section tarifs | 0 EUR |
| scroll_depth_75 | Scroll 75% landing page | 0 EUR |

### Formats Display

- **Banner responsive** : Titres + descriptions + images (fournis automatiquement)
- **Image ads** : 300x250, 728x90, 160x600
