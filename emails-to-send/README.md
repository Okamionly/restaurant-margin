# 📧 Emails prêts à envoyer

7 emails pré-rédigés pour ton plan link building 30 jours.

## Comment envoyer

### Option 1 : Mail client (Outlook, Apple Mail, Thunderbird)
**Double-clique** sur le `.eml` → ton mail client s'ouvre avec l'email pré-rempli → vérifie + clique "Envoyer".

### Option 2 : Gmail web
1. Ouvre Gmail → "Composer un nouveau message"
2. Ouvre le `.eml` avec un éditeur de texte
3. Copie/colle dans l'ordre :
   - **À** : ligne `To:`
   - **Objet** : ligne `Subject:`
   - **Corps** : tout ce qui est en dessous de `Content-Type: text/plain; charset=utf-8`
4. Vérifie tes coordonnées (téléphone) puis Envoie
